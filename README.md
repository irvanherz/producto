# PRODUCTO

A very-very simple product management, built with React + Redux.

## Getting Started

- **Clone**
  ```
  git clone https://github.com/irvanherz/producto.git
  ```
- **Run development server**
  ```
  cd producto
  npm run dev
  ```
  or **Build**
  ```
  npm run build
  ```

- **Deploy via FTP**
  ``` 
  cp .env.template .env
  nano .env #setup as needed
  npm run deploy
  ```
